Larger Text Tag

For larger text use <big> tag on your HTML page.

<big> tag: “used to display a larger text on an HTML page”

Syntax: 

<big>
//content
</big>

Using this <big> tag we can display a larger text on our HTML page. All the content in between this tag appears to be displayed one font size larger. <big> tag has also its corresponding >/big> tag.

NOTE: According to HTML5, this tag has been deprecated and is now obsolete. We'll learn more about deprecated tags in the upcoming tutorial.
