Strike Text Tag

For strike text use <strike> tag on your HTML page.

<strike> tag: “used to make a thin line through text on an HTML page”

Syntax: 

<strike>
//content
</strike>

Using this <strike> tag we can make a thin line on our text. All the content in between this tag appears to be displayed with a strikethrough. <strike> tag has also its corresponding </strike> tag.

NOTE:According to HTML5, this tag has been deprecated and is now obsolete. We'll learn more about deprecated tags in the upcoming tutorial.
