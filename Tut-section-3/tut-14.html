Smaller Text Tag

For smaller text use <small> tag on your HTML page.

<small> tag: “used to display a smaller text on an HTML page”

Syntax: 

<small>
//content
</small>

Using this <small> tag we can display a smaller text on our HTML page. All the content in between this tag appears to be displayed one font size smaller. <small> tag has also its corresponding </small> tag.
